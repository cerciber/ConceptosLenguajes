
# Enviar cambios al origin de la rama actual

# Enviar cambios al master
git push origin master

# Enviar cambios a una rama especifica
git push origin rama

# Enviar cambios a la rama remota de la rama actual
git push

git log (Para ver la referencia de los commits) (q: para salir de la vista)
git revert referencia_commit (E: para sobreescribir si hay conflictos) (:qa para salir del mensaje)
